# Trouve ton artisan

Ce site web est un projet qui a pour objectif de **permettre aux particuliers de trouver un artisan proche de chez eux**.
On peut rechercher des artisans par catégories, spécialités ou noms.

Projet réalisé avec un backend API qui interroge une base de donnée MySQL et un frontend REACT.


## Prérequis

Avant de commencer, assurez-vous d’avoir installé :
- [Node.js](https://nodejs.org/)
- [XAMPP](https://www.apachefriends.org/fr/index.html) (pour MySQL)

## Guide d'installation

### Base de données MySQL
- Lancer **XAMPP**
- Démarrer **Apache** et **MySQL**
- Créer la base de données via **phpMyAdmin**

### Installation de backend
cd backend
npm install
npm run dev

Backend accessible à l'adresse : http://localhost:5000

### Installation du frontend
cd frontend
npm install
npm run dev

Frontend accessible à l'adresse : http://localhost:5173
